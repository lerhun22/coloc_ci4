# ✅ 4 — Ajouter login legacy dans importer

Au début de importCompetition :

```php
use App\Libraries\CopainLegacyReader;
```

Puis :

```php
public function importCompetition($ref, $type, $ordre)
{

    /*
    ======================
    LOGIN LEGACY OBLIGATOIRE
    ======================
    */

    $legacy = new CopainLegacyReader();

    $login = $legacy->getCompetitions(
        $this->email,
        $this->password
    );

    if (!$login || $login['code'] != 0) {

        return [
            'code' => 'LOGIN_ERROR',
            'login' => $login
        ];
    }
```

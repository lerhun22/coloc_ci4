# 📊 Base de données

## Table `competitions`

* id
* nom
* date_competition (DATE NOT NULL)
* saison
* date_creation

## Table `photos`

* id
* competitions_id
* titre
* note_totale
* place
* etc.

`nb_oeuvres` est calculé via :

```sql
COUNT(photos.id)
GROUP BY competitions.id
```

Jointure LEFT JOIN dans le controller.
